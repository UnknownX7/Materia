# Materia
Materia is a framework for creating plugins for [FINAL FANTASY VII EVER CRISIS](https://store.steampowered.com/app/2484110/FINAL_FANTASY_VII_EVER_CRISIS/). Heavily inspired by [Dalamud](https://github.com/goatcorp/Dalamud).

Currently in alpha testing, you can find information via the [Discord](https://discord.gg/fVGV6tbr5J).
